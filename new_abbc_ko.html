ABBC 구매 방식이 바뀌었나요?

네, 마일리지(₩)로 ABBC를 구매했던 방식은 2021년 4월 30일부로 종료되었습니다.

바이알라딘은 ABBC 코인 탑재를 완료하였으며, 거래소 또는 사용자 암호화폐 지갑에 보유한 ABBC 코인을 전송하여 이용하실 수 있으니 착오 없으시기 바랍니다.

바이알라딘 ABBC 지갑에 코인 ‘받기’ 방법

사용자 암호화폐 지갑에서 보유하고 있는 ABBC를 바이알라딘 앱 내 ABBC 카드에 보낸 후 사용하실 수 있습니다. ABBC 가치는 실시간 시장 가격을 기준으로 바이알라딘이 책정합니다. 또한 시세는 지속적으로 달라질 수 있습니다.

입금 단위는 0.00000001ABBC이며 무제한으로 보낼 수 있습니다. 기존 ABBC와 같이 소수점 아래 8자리까지 표기됩니다. 사용자 암호화폐 지갑에서 보낸 후 사용할 수 있으며, 보내기가 성공적으로 완료되면 바이알라딘 앱 내 ABBC 카드에 사용자가 보유한 수량이 보여집니다.

바이알라딘이 책정한 최소 수량 이상 보내 주시기 바랍니다. 최소 수량 이하로 보낼 경우, 정상적으로 ‘받기’가 어려울 수 있습니다.

‘받기’ 이용 가이드를 알고 싶습니다.

아래 가이드를 참고해 주세요.


My Wallet에서 ‘받기’ 버튼을 클릭해 주세요.

서비스 이용 동의 및 가이드를 확인 후, ‘동의’ 버튼을 클릭해 주세요.

암호화폐 서비스 이용약관을 확인 후, ‘동의’ 버튼을 클릭해 주세요.

처음 이용하시는 경우, Passcode를 설정해 주세요.

Passcode 설정이 완료되면 사용자의 BTC, ETH, ABBC 지갑 주소가 각각 생성됩니다. 받은 지갑 주소를 복사하여 보관해 주세요.

My Wallet 페이지에서 한 번 더 ‘받기’ 버튼을 클릭해 주세요.

‘받기 요청서’ 페이지에서 보내실 경로를 선택해 주세요. (지갑 또는 거래소 택1)

보낼 날짜, 시간, 수량을 입력 후, ‘OK’ 버튼을 클릭해 주세요. (최소 수량 이하로 보낼 경우, 정상적으로 ‘받기’가 어려울 수 있습니다.)

Wallet QR코드 화면으로 이동됩니다.

고객님이 이용중인 거래소 또는 개인지갑에서 Wallet QR코드 또는 주소를 참고하여 송금해 주세요.

송금이 완료되면 보낸 수량이 지갑에 노출됩니다. 바이알라딘 My Wallet 페이지 최근 거래 내역을 확인해 주세요.

‘받기’가 완료되면 등록된 사용자 이메일로 ‘받기 완료’ 이메일이 발송되며, 앱 내의 알림으로 확인 가능합니다.

바이알라딘 ABBC 지갑에서 ABBC ‘보내기’ 방법

‘보내기’는 바이알라딘의 승인 절차에 의해 외부지갑으로 전송됩니다.


‘보내기’ 요청 시, 수수료가 청구됩니다. 보유 수량이 수수료를 포함한 보낼 수량보다 적을 경우 ‘보내기’가 불가합니다.


‘보내기’ 요청 시, 승인 절차로 인해 최소 24시간에서 최대 48시간까지 소요될 수 있습니다.


승인이 완료된 후, 자동 시스템에 의해 요청하신 수량만큼 입력된 지갑 주소로 보내 집니다.

ABBC ‘보내기’ 이용 가이드를 알고 싶습니다.

아래 가이드를 참고해 주세요.


바이알라딘 카드 내 ABBC를 사용자 암호화폐 지갑으로 보내기 위해 ‘보내기’ 버튼을 클릭해 주세요.

‘보내기’ 요청 시, 바이알라딘의 승인 절차를 따르므로 ‘보내기 승인’에 ‘동의’ 버튼을 클릭해 주세요.

보낼 사용자 암호화폐 지갑 주소를 입력해 주세요. (QR코드로 입력 가능)

보낼 수량을 입력해 주세요. 입력된 수량에 대한 달러가치는 자동계산 되어 표기됩니다. 이용가능한 ABBC는 외부지갑에 '보내기' 할 경우 시세와 동일하게 계산됩니다.

이메일 주소 입력 후, 인증을 완료해 주세요. 이메일은 바이알라딘에 등록한 이메일로만 인증이 가능합니다.

‘보내기’ 버튼을 클릭 시, 요청한 수량만큼 입력된 지갑 주소로 보내집니다.

ABBC ‘보내기’ 후에도 재화 이용이 가능한가요?

보내기 처리가 완료되기 전까지 추가 ‘보내기’ 또는 상품구매가 불가합니다.
ABBC ‘보내기’ 시, ABBC ‘받기’ 및 타 재화는 정상 이용 가능합니다.

ABBC 사용 방법을 알고 싶습니다.

바이알라딘 지갑에 ABBC를 보유하여 전환된 이용가능한 ABBC로 바이알라딘에 판매자들이 등록한 다양한 상품을 구매할 수 있습니다.
바이알라딘 내에서 이용가능한 ABBC로 상품 구매 시 1UABBC당 $100 의 가치를 지니며 다른 결제수단과 복합 결제는 불가합니다.
