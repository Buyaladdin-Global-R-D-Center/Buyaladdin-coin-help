바이알라딘 내 바이낸스코인(BNB)이 추가된 이유가 있나요?

바이알라딘은 글로벌 통합 결제 플랫폼 구축을 위해 바이낸스(BNB) 결제 방식을 도입하였습니다.
전세계 수백만명의 사람들이 국가에 제한 없이 편리하고 간편하게 상품을 구매할 수 있도록 시스템 구축에 힘쓰고 있습니다.

바이알라딘 BNB 지갑에 어떻게 받을 수 있나요?

사용자 암호화폐 지갑에서 보유하고 있는 바이낸스코인(BNB)을 바이알라딘 앱 내 BNB카드에 보낸 후 사용하실 수 있습니다.
바이낸스(BNB) 가치는 실시간 시장 가격을 기준으로 바이알라딘이 책정합니다. 또한 시세는 지속적으로 달라질 수 있습니다.


입금 단위는 (0.00000001변경가능)BNB이며 무제한으로 보낼 수 있습니다. 기존 비트코인과 같이 소수점 아래 8자리까지 표기됩니다.
사용자 암호화폐 지갑에서 보낸 후 사용할 수 있으며, 보내기가 성공적으로 완료되면 바이낸스(BNB)카드에 사용자가 보유한 수량이 보여집니다.


바이알라딘이 책정한 최소 수량 이상 보내 주시기 바랍니다. 최소 수량 이하로 보낼 경우, 정상적으로 ‘받기’가 어려울 수 있습니다.

‘받기’ 이용 가이드를 알고 싶습니다.

아래 가이드를 참고해 주세요.


Wallet에서 ‘받기’ 버튼을 클릭해 주세요.

서비스 이용 동의 및 가이드를 확인 후, ‘동의’ 버튼을 클릭해 주세요.

암호화폐 서비스 이용약관을 확인 후, ‘동의’ 버튼을 클릭해 주세요.

처음 이용하시는 경우, Passcode를 설정해 주세요.

Passcode 설정이 완료되면 사용자의 BTC, ETH, ABBC 지갑 주소가 각각 생성됩니다. 받은 지갑 주소를 복사하여 보관해 주세요.

Wallet 페이지에서 한 번 더 ‘받기’ 버튼을 클릭해 주세요.

‘받기 요청서’ 페이지에서 보내실 경로를 선택해 주세요. (지갑 또는 거래소 택1)

보낼 날짜, 시간, 수량을 입력 후, ‘OK’ 버튼을 클릭해 주세요. (최소 수량 이하로 보낼 경우, 정상적으로 ‘받기’가 어려울 수 있습니다.)

Wallet QR코드 화면으로 이동됩니다.

고객님이 이용중인 거래소 또는 개인지갑에서 Wallet QR코드 또는 주소를 참고하여 송금해 주세요.

송금이 완료되면 보낸 수량이 지갑에 노출됩니다. 바이알라딘 Wallet 페이지 최근 거래 내역을 확인해 주세요.

‘받기’가 완료되면 등록된 사용자 이메일로 ‘받기 완료’ 이메일이 발송되며, 앱 내의 알림으로 확인 가능합니다.

보유 바이낸스(BNB)을 제 지갑으로 다시 보낼 수 있나요?

네, ‘보내기’는 바이알라딘의 승인 절차에 의해 외부지갑으로 전송됩니다.


‘보내기’ 요청 시, 수수료가 청구됩니다. 보유 수량이 수수료를 포함한 보낼 수량보다 적을 경우 ‘보내기’가 불가합니다.


‘보내기’ 요청 시, 승인 절차로 인해 최소 24시간에서 최대 48시간까지 소요될 수 있습니다.


승인이 완료된 후, 자동 시스템에 의해 요청하신 수량만큼 입력된 지갑 주소로 보내 집니다.

‘보내기’ 이용 가이드를 알고 싶습니다.

아래 가이드를 참고해 주세요.


바이알라딘 카드 내 BNB를 사용자 암호화폐 지갑으로 보내기 위해 ‘보내기’ 버튼을 클릭해 주세요.

‘보내기’ 요청 시, 바이알라딘의 승인 절차를 따르므로 ‘보내기 승인’에 ‘동의’ 버튼을 클릭해 주세요.

보낼 사용자 암호화폐 지갑 주소를 입력해 주세요. (QR코드로 입력 가능)

보낼 수량을 입력해 주세요. 원화는 자동계산 되어 표기됩니다.

이메일 주소 입력 후, 인증을 완료해 주세요. 이메일은 바이알라딘에 등록한 이메일로만 인증이 가능합니다.

‘보내기’ 버튼을 클릭 시, 요청한 수량만큼 입력된 지갑 주소로 보내집니다.

‘보내기’ 후에도 재화 이용이 가능한가요?

네, 가능합니다. BNB ‘보내기’ 시, 보내기 처리가 완료되기 전까지 추가 ‘보내기’ 또는 상품구매가 불가합니다. (BNB ‘보내기’ 시, ‘받기’ 기능 및 타 재화는 정상 이용 가능합니다.)
